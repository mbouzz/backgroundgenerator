# Background Gradient Generator

A simple web project that allows you to dynamically generate CSS gradient backgrounds.

---

## Features

* Select two colors using color pickers
* Automatically generate a CSS gradient
* Display the current CSS background code in real time

---

## Project Structure

```
.
├── index.html
├── style.css
└── script.js
```

---

## Preview

Users can select two colors and instantly see the result as the page background, along with the corresponding CSS code.

---

## Technologies Used

* HTML5
* CSS3 (linear-gradient)
* JavaScript (DOM manipulation)

---

## Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
```

2. Open the `index.html` file in your browser.
